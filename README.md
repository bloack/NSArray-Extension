## Installation

NSArray-Extension is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'NSArray-Extension',:git=>'git@github.com:bloack/NSArray-Extension.git'
```
